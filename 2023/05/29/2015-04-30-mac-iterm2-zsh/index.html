Mac Iterm2 zsh使用

罗昭成 2023年05月29日

安装zsh

brew install zsh

下载oh my zsh项目

git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh

设置zsh

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
chsh -s /bin/zsh

VIM 的一些配置

syntax on                "语法高亮
set nu!                  "显示行号
set ai!                  "设置自动缩进
set ruler                "在编辑过程中，在右下角显示光标位置的状态行
set incsearch            "自动匹配单词的位置
set hlsearch             "高亮查找到的所有单词
